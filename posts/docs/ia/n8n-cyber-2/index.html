Dans la suite de mon dernier article. On va integrer VirusTotal a notre agent n8n pour analyser des urls.

Je vous rappelle le but de notre agent qui va nous servir a analyser les urls des sites louches que je croise depuis mon telephone en envoyant un simple message telegram. Donc il me repondra avec des informations sur le site. Et les conseils specifiques pour ce site. Donc un genre de majordome virtuel qui va me dire si y aller ou pas.

Integration de VirusTotal

Pour ceux qui ne connaissent pas VirusTotal, c’est un service qui permet d’analyser des fichiers et des urls pour detecter des malwares. Donc il va envoyer ses urls ou fichiers a plusieurs antivirus pour les analyser. Et on aura une interface web unifie pour voir les resultats.

Par exemple :

Donc il nous donne un score, qui signifie le nombre de moteurs d’antivirus qui ont detecte un malware. Donc plus le score est eleve, plus c’est plaussible qu’il soit dangereux.

Pour n8n, ils ont deja un module pour VirusTotal. Pour l’utiliser, toujours pareil, on va aller dans le petit plus en haut a droute, puis on va chercher VirusTotal. Et on va le connecter a notre agent.

Ensuite vous aurez ce menu qui va s’afficher :

La partie qui nous interesse ici, c’est Credential for VirusTotal. VirusTotal va nous demander une clef API pour pouvoir utiliser leur service. Pour creer cette clef, tout a fait gratuite, mais avec des restrictions, il faut creer un compte sur leur site. Et cliquer sur API Key

Une fois dedans vous pourrez creer une clef API. Et vous la copiez dans le champ presente tout a l’heure.
